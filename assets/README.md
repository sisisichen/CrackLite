# Assets

This directory contains GitHub-friendly PNG assets used by the README.

```text
figures/
  fig2_architecture.svg
  fig3_cracklite_block.svg
  fig4_dataset_miou.svg
  fig5_qualitative_comparison.png
  fig6_internal_response.png
  fig7_morphology_pipeline.svg
  fig8_efficiency_tradeoff.svg
  fig9_ablation_miou.svg
  fig10_ablation_qualitative.png
```

Vector SVG files are used for structure diagrams and charts so they remain
sharp on GitHub. PNG files are used for dense qualitative matrices where SVG
exports would be unnecessarily large. The editable source drawings remain
outside the repository. Large editable
source files such as `.vsdx`, `.opju`, or manuscript drafts should not be
committed unless they are intentionally released.
