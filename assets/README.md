# Assets

This directory contains GitHub-friendly PNG assets used by the README.

```text
figures/
  fig1_performance_complexity.png
  fig2_architecture.png
  fig3_cracklite_block.png
  fig4_dataset_miou.png
  fig5_qualitative_comparison.png
  fig6_internal_response.png
  fig8_efficiency_tradeoff.png
```

These PNG files were exported from the manuscript figure source files so they
render reliably on GitHub. The editable source drawings remain outside the
repository. Large editable source files such as `.vsdx`, `.opju`, or manuscript
drafts should not be committed unless they are intentionally released.
